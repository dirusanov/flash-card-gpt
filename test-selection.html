Test Text Selection for Anki Extension

Тестовые случаи для проверки исправлений:

1. Короткий текст (3 слова) - должен использоваться автоматически:

Выделите: beautiful sunny day

2. Текст из 4-5 слов - должны предлагаться варианты:

Выделите: the beautiful sunny morning weather

Ожидаемые варианты: полная фраза + отдельные слова (beautiful, sunny, morning, weather)

3. Текст из 6-7 слов - должны предлагаться варианты:

Выделите: we enjoyed the beautiful sunny morning together

Ожидаемые варианты: полная фраза + отдельные слова + комбинации из 2 слов

4. Средний текст из 8-10 слов:

Выделите: the magnificent architecture of the ancient building was truly remarkable

Ожидаемые варианты: полная фраза + отдельные слова (magnificent, architecture, ancient, building, remarkable) + комбинации

5. Текст со сложными словами:

Выделите: extraordinary multifaceted comprehensive analysis

Ожидаемые варианты: все слова должны быть включены как отдельные варианты

6. Текст с разными частями речи:

Выделите: quickly running through the forest

Ожидаемые варианты: полная фраза + важные слова (quickly, running, through, forest)

Инструкции для тестирования:

  1. Установите обновленное расширение в Chrome
  2. Выделите каждый из тестовых текстов
  3. Проверьте, что для текстов 4+ слов появляется модальное окно с вариантами
  4. Убедитесь, что предлагаются как отдельные слова, так и фразы
  5. Проверьте, что для текстов 3 слова и меньше модальное окно не появляется
