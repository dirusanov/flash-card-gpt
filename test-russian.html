Тест русского текста для Anki расширения

Тестовые случаи для проверки исправлений:

1. Короткий текст (3 слова) - автоматическое использование:

Выделите: красивый солнечный день

Ожидается: автоматическое использование без модального окна

2. Текст из 4 слов - должны предлагаться варианты:

Выделите: красивая солнечная погода сегодня

Ожидается: полная фраза + слова (красивая, солнечная, погода, сегодня)

3. Текст из 5-6 слов - ваш исходный пример:

Выделите: из-за приспособления к разным средам

Ожидается: полная фраза + слова (приспособления, разным, средам) + комбинации

4. Сложные русские слова:

Выделите: необыкновенное преобразование окружающей среды

Ожидается: полная фраза + все значимые слова отдельно

5. Научный текст:

Выделите: биологические адаптационные механизмы организма

Ожидается: полная фраза + слова (биологические, адаптационные, механизмы, организма)

6. Смешанный текст:

Выделите: современные технологии artificial intelligence

Ожидается: обработка и русских, и английских слов

7. Текст с предлогами (должны фильтроваться):

Выделите: изучение влияния на развитие

Ожидается: слова (изучение, влияния, развитие), предлог "на" должен быть отфильтрован

Инструкции для тестирования:

  1. Установите обновленное расширение в Chrome
  2. Откройте Developer Tools (F12) и перейдите на вкладку Console
  3. Выделите каждый из тестовых текстов
  4. Проверьте логи в консоли:
    • Extracted words from text: - какие слова извлечены
    • Filtered words: - какие слова прошли фильтрацию
    • Added individual word: - какие слова добавлены в опции
  5. Убедитесь, что в модальном окне появляются отдельные слова

Дополнительные короткие тесты:

Выделите: книга (1 слово)

Выделите: хорошая книга (2 слова)

Выделите: очень хорошая книга (3 слова)

Выделите: очень хорошая интересная книга (4 слова - должно показать варианты)
